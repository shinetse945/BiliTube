### This project is a fork built on top of [Danmaku-Kakashi-Ext](https://github.com/Danmaku-Kakashi/Danmaku-Kakashi-Ext)

### Enjoy danmaku (bullet comments) on YouTube™

##### Features

- Search for matching BiliBili videos from within YouTube

- Load BiliBili danmaku onto YouTube videos

##### How to Use

1. Download and extract BiliTube-vX.X.zip.

2. Open chrome://extensions/ and enable "Developer mode".

3. Click "Load unpacked" in the top-left corner.

4. Select the extracted folder.

Do not delete the extracted folder, otherwise the extension will stop working!

##### Technologies Used

- React.js

- Node.js

- Chrome API

- Material UI (MUI)

- [Comment Core Library](https://github.com/jabbany/CommentCoreLibrary) - Licensed under the MIT license
