### 本项目为在[Danmaku-Kakashi-Ext](https://github.com/Danmaku-Kakashi/Danmaku-Kakashi-Ext)基础上二次开发的项目

### 在 YouTube™上享受弹幕

##### 功能

- 在 YouTube 上搜索匹配的 BiliBili 视频 

- 加载 BiliBili 视频弹幕到 YouTube 

##### 如何使用

1、下载并解压BiliTube-v某某.zip。

2、打开 chrome://extensions/，开启 “开发者模式”。

3、点击左上角的 “加载未打包的扩展程序”。

4、选择解压后的文件夹即可。

解压后的文件夹不可删除，否则插件会失效！


##### 使用的技术

- React.js

- Node.js

- Chrome API

- Material UI (MUI)

- [Comment Core Library](https://github.com/jabbany/CommentCoreLibrary) - Licensed under the MIT license
